String Trimming Function

 Function:
GS.trim(<STRING-TO-BE-TRIMMED>, <STRING-TO-TRIM-OFF>);
Description:
The GS.trim function is for cutting off characters at the beginning and end of a string.
Examples:
Skeleton Example:
When you click the button the text of the first text control will have the text of the second text control trimmed off of it and the result will be put in the second text control.
