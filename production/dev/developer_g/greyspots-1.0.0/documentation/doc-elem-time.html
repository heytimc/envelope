Time

 Tag:
<gs-time></gs-time>
Description:
The gs-time control is like the gs-text control, but it is specialized for times.
Examples:
Skeleton Example:
A basic gs-time element. Without attributes the gs-time control will have the following features/behaviors:
    1) There is a button that activates the time picker.
    2) If the time picker is in modal mode: the time picker can be dismissed by clicking the "Done" button. If the time picker is not in modal mode: clicking out of the time control will dissmiss the time picker.
    3) You can also change the time manually.
Value Attribute Example:
To set the value for the gs-time, use the "value" attribute.
Query String Attribute Example:
The "qs" attribute can be used to sync the gs-date with the query string.
Placeholder Attribute Example:
The "placeholder" attribute specifies some placeholder text for when the control is empty.
Mini Attribute Example:
Sometimes you are limited on space. The mini attribute is used to make the gs-time smaller.
No Picker Attribute Example:
If you want to remove the date picker button: add the "no-picker" attribute.
No Now Button Attribute Example:
If you want to remove the "Now" button from the picker: add the "no-now-button" attribute.
Non-Empty Attribute Example:
If you want to force the user to input a value: add the non-empty attribute. Note: if a default value is not provided then the control defaults to "12:00 PM".
Display Format Attribute:
The "format" attribute determines the display format of the control. This attribute can take two different values: "regular" and "military". "regular" is normal 12-hour time and "military" is 24-hour time.
Output Format Attribute Example:
The "output-format" attribute determines the internal format of the control. This attribute can take two different values: "regular" and "military". "regular" is normal 12-hour time and "military" is 24-hour time. When getting the value of the control in javascript "output-format" is the format it will get.
Tabindex Attribute Example:
With the "tabindex" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example:
With the "disabled" attribute, you can control whether or not you can use the control.
