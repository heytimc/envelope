Search

 Tag:
<gs-search></gs-search>
Description:
The gs-search control allows you to alter the query string for searching. This is used in conjuction with the qs attribute on other controls.
Database Functionality For Examples:
Examples on this page require the "gsdoc" schema to work properly. To enable the demonstration of features that require database functionality, please click the following button to create or refresh the "gsdoc" schema.
The "gsdoc" schema is only installed temporarily. It is lost after every Abandon, Pull or Release.
Create/Refresh Documentation Schema
Examples:
Skeleton Example:
Without attributes the gs-search will have the following behaviors:
    1) You can type into the control.
    2) The placeholder will default to "Search...".
    3) It won't change the query-string because we haven't told it what parameter to change.
QS Attribute:
In order to change the query string using the text inside the control, you need to use the qs="" attribute.
Placeholder Attribute Example:
The "placeholder" attribute specifies some placeholder text for when the control is empty.
Mini Attribute Example:
Sometimes you are limited on space. The mini attribute can be used to make the gs-search smaller.
Tabindex Attribute Example:
With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example:
With the disabled attribute, you can control whether or not you can use the control.

This is an example of how you would use the <gs-search> for filtering.
