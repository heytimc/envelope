Ajax Error Dialog Function

 Tag:
GS.ajaxErrorDialog(data, function (data, error) {
    ...
}, function (data, error) {
    ...
});
Description:
Examples:
Skeleton Example: A basic error dialog. the will have the following behaviors:
    1) It will show the user an error.
    2) It will allow them to try again.
No Callback Example: This will not allow the user to try again, because there is no callback passed to the function.
