GUID Function

 Function:
GS.GUID();
Description:
This function returns a randomly generated GUID. GUID stands for "Globally Unique IDentifier" what this means is that the function will return a string of random letters and numbers that will most likely never be repeated on a webpage twice ever.
Examples:
GS.GUID Example:
In this example when you click the "Test GUID" button a randomly generated GUID will be put into the "result" div.
