Optionbox

 Tag:
<gs-optionbox>
    <gs-option>...</gs-option>
</gs-optionbox>
Description:
A gs-optionbox control allows you to select an option from a list.
Examples:
Skeleton Example: A basic gs-optionbox element with six options. Without attributes the optionbox will have the following behaviors:
    1) Every other option will have a different color to help differentiate between options.
    2) Clicking on an option will select it, changing its color.
Option Value Attribute Example: When you get the value of a gs-optionbox, it will use the value="" attribute of the gs-option to return. If there is no value="" attribute on the gs-option, it will return the text of the gs-option.
Optionbox Value Attribute Example: To set the default value for the gs-optionbox, use the value="" attribute.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-optionbox with the query string.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
