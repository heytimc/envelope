Select

 Tag:
<gs-select>
    <option>...</option>
</gs-select>
Description:
The gs-select allows you to choose from a static dropdown list of options.
Examples:
Skeleton Example: Without attributes the gs-select will have the following behaviors:
    1) The first option is the default.
    2) Clicking on the control brings up a list of options.
    3) Clicking on the a option in the control closes the list and changes the option the control has selected.
Option Value Attribute Example: When you get the value of a gs-select, it will use the value="" attribute of the gs-option to return. If there is no value="" attribute on the gs-option, it will return the text of the gs-option.
Optionbox Value Attribute Example: To set the default value for the gs-select, use the value="" attribute.
Mini Attribute Example: The mini attribute allows you to fit the gs-select in tighter spaces.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-select with the query string.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
