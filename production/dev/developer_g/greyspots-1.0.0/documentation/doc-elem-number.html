Number

 Tag:
<gs-number></gs-number>
Description:
The purpose of a gs-number control to take hold a number from the user. This control is based off of the gs-text.
Examples:
Skeleton Example: A basic gs-number element. Without attributes the gs-number control will have the following behaviors:
    1) You can change the number manually.
    2) Without attributes it's kinda boring.
Value Attribute Example: To set the default value for the gs-number, use the value="" attribute.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-number with the query string.
Placeholder Attribute Example: The placeholder="" attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)
Mini Attribute Example: Sometimes you are limited on space. The mini attribute can be used to make the gs-number smaller.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
