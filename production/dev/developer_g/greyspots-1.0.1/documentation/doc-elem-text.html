Text

 Tag:
<gs-text></gs-text>
Description:
Examples:
Skeleton Example: Without attributes the gs-text will have the following behaviors:
    1) Clicking on the control gives you a cursor.
    2) You can change the text bu typing.
Value Attribute Example: To set the default value for the gs-text, use the value="" attribute.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-text with the query string.
Placeholder Attribute Example: The placeholder="" attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)
Mini Attribute Example: Sometimes you are limited on space. The mini attribute can be used to make the gs-text smaller.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Autocorrect/Autocapitalize/Autocomplete Attribute Example: If you don't want spelling to autocorrect, you can use the html5 autocorrect="off" attribute. See also html5 autocapitalize="off" and autocomplete="off".
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
