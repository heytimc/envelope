Cookie Functions

 Functions:
GS.setCookie(<COOKIE-NAME>, <NEW-VALUE>, <NUMBER-OF-DAYS-UNTIL-EXPIRATION>);
GS.getCookie(<COOKIE-NAME>);
Description:
These functions are for getting and setting browser cookies. For documentation about cookies themselves: MDN, W3Schools.
Examples:
GS.setCookie And GS.getCookie Example:
In this example you set the value of the "doc_cookie" by filling in the "text-cookie-new-value" control and clicking "Set Cookie". Then you can get the value back from the cookie by clicking the "Get Cookie" button. Notice: if you set the cookie then refresh the page then click the "Get Cookie" button the value will still be there.
