How To Add Links To The Main Menu
Open the "File Manager"
Navigate to the "all" folder
Open the "File Editor" for the "index.html" file
Find the text '<div id="user-app-list-container">'
Inside the "user-app-list-container" div element create a new <gs-button> element
Fill in the text of the button
Add the "target" attribute and set it to "_blank"
Add an "href" attribute that links to your app's main page
If you want to resrict the button to a certain group or user:
    To restrict to a group: set the "data-group" attribute to what group you want to see it
    To restrict to a user: set the "data-user" attribute to what user you want to see it

Done.
