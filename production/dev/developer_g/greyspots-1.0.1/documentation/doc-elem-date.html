Date

 Tag:
<gs-date></gs-date>
Description:
The gs-date control is like the gs-text control, but it is specialized for dates.
Examples:
Skeleton Example: A basic gs-date element. Without attributes the gs-date control will have the following behaviors:
    1) There is a button that activates the date picker.
    2) After choosing a date using the date picker, the date picker disappears and the data changes in the control.
    3) You can also change the date manually.
Value Attribute Example: To set the default value for the gs-date, use the value="" attribute.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-date with the query string.
Placeholder Attribute Example: The placeholder="" attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)
Mini Attribute Example: Sometimes you are limited on space. The mini attribute can be used to make the gs-date smaller.
No Picker Attribute Example: If you want to remove the date picker button, simply add the no-picker attribute.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
