Loader

 Function:
GS.addLoader
Description:
This function puts a modal spinner with optional text over an element or over the whole document. To remove a loader use the function GS.removeLoader (for more information on the use of GS.removeLoader: see examples).
Parameters:
# Name Input Optional Description
1 Loader Class or Target A CSS class (covers the viewport) or
An HTML element (covers an element)		 Yes This parameter is for telling the function where to put the loader. It is also for being able to later remove the loader.
2 Loader Text HTML String Yes This is the content of the loader.
Skeleton Example:
GS.addLoader('loading-contacts', 'Loading...');
