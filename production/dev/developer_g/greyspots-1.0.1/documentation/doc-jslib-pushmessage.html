Pushmessage

 Function:
GS.pushMessage(...);
Description:
This Javascript function puts a temporary message with HTML in it at the top of the document.
Examples:
Skeleton Example: A basic pushmessage with some text in it. The message will have the following behaviors:
    1) The first one will appear at the top of the screen. Subsequent messages will appear at the bottom of the list of messages.
    2) The messages will be permanent because we are not closing them.
Timed Closing Example: A pushmessage with some text in it and a time. These messages will close themselves after 1.5 seconds.
Manual Closing Example: A button to open a pushmessage and a button to close a pushmessage.
Dynamic Message Example: A pushmessage with some text in it, a time and a callback function. These messages will have a number that increments inside.
