Account Functions

 Functions:
GS.userChangePassword();
GS.superChangePassword();
GS.superUserLogin(<LOGGED-IN-CALLBACK>);
Description:
These functions handle some account-related operations.
Warning:
The first two examples on this page will change this account's passwords if you allow them, please use caution.
Examples:
GS.userChangePassword Example:
In this example when you click the "" button the GS.userChangePassword function will open a dialog that will allow you to change your normal user password.
GS.superChangePassword Example:
In this example when you click the "" button the GS.superChangePassword function will open a dialog that will allow you to change your super user password.
GS.superUserLogin Example:
This function will check to see if you are logged in as a superuser. If you are not logged in as a superuser: the function will open a dialog so that you can log in as a superuser then when you successfully log in: the callback will be called. If you are already logged in as a superuser it will just call the callback.

Note: You have to be logged in as a superuser to load this documentation page so unless you log out in a seperate window: you will just go straight to the callback.
