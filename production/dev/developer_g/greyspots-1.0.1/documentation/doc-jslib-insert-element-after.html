Insert Element After Function

 Function:
GS.insertElementAfter(<ELEMENT-TO-INSERT>, <TARGET>)
Description:
This function fills in a missing Javascript DOM manipulation function: inserting an element after another element. Currently Javascript can append an element into another and insert an element before another, but not insert after.
Examples:
GS.insertElementAfter Example:
When you click the "Insert Element" button a div element with some text inside will be inserted right after the button.
