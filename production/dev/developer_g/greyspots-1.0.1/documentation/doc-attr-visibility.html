Visibility Attributes

 Attributes:
show-on-desktop
show-on-tablet
show-on-phone
hide-on-desktop
hide-on-tablet
hide-on-phone
Description:
These attributes are for making elements appear and disappear depending on screen size.
    show-on-desktop (only show at >= 769px)
    show-on-tablet  (only show at >= 321px)
    show-on-phone   (only show at <= 321px)
    hide-on-desktop (only hide at >= 769px)
    hide-on-tablet  (only hide at >= 321px)
    hide-on-phone   (only hide at <= 321px)
Examples:
All Attributes Example:
In this example we have a list of divs each with one of the hide or show attributes. In each div are some numbers and the name of the attribute that was used on it. The numbers are just to show at what sizes an element would be visible.
    1 means visible at phone size
    2 means visible at tablet size
    3 means visible at desktop size
