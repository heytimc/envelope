Static

 Tag:
<gs-static></gs-static>
Description:
The gs-static allows you to view data from a gs-form without the ability to change that data.
Database Functionality For Examples:
Examples on this page require the "gsdoc" schema to work properly. To enable the demonstration of features that require database functionality, please click the following button to create or refresh the "gsdoc" schema.
The "gsdoc" schema is only installed temporarily. It is lost after every Abandon, Pull or Release.
Create/Refresh Documentation Schema
Examples:
Skeleton Example: A basic gs-static control will have the following behaviors:
    1) The value attribute will set the text of the control.
    2) The control cannot be focused.
    3) The text can be selected so that it can be copied.
Pre Attribute Example: Use the pre attribute to format the text. Note that a "\n" in the value attribute will show as "\n", but a newline in the attribute text will show properly.
Query String Attribute Example: The qs="" attribute can be used to sync the gs-static with the query string.
Inline Attribute Example: Sometimes you are limited on space. The inline attribute can be used to make the gs-static smaller width-wise.
Mini Attribute Example: Sometimes you are limited on space. The mini attribute can be used to make the gs-static smaller height-wise.
