Search

 Tag:
<gs-search></gs-search>
Description:
The gs-search control allows you to alter the query string. This is used in conjuction with the qs attribute on other controls.
Examples:
Skeleton Example: Without attributes the gs-search will have the following behaviors:
    1) You can type into the control.
    2) The placeholder will default to "Search...".
    3) It won't change the query-string because we haven't told it what parameter to change.
QS Attribute: In order to change the query string using the text inside the control, you need to use the qs="" attribute.
Placeholder Attribute Example: The placeholder="" attribute can be used when you don't want to use a label to explain the purpose of the control to the user. (The label and the placeholder can be used together, that's just not what normally happens.)
Mini Attribute Example: Sometimes you are limited on space. The mini attribute can be used to make the gs-search smaller.
Tabindex Attribute Example: With the tabindex="" attribute, you can control what order your controls are tabbed to.
Disabled Attribute Example: With the disabled attribute, you can control whether or not you can use the control.
