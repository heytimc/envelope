String To Element Function

 Function:
GS.stringToElement(<HTML-STRING>, <OPTIONAL-DOCUMENT>);
Description:
The GS.stringToElement function is for creating an HTML element based on the element's string representation.
Examples:
GS.stringToElement Example:
In this example when you click the "Create/Append Element" button an element will be generated from a string and appended to the "target" div. Note: If you try to do two elements next to each other at the top level then you will only get the first element, however put several elements inside the first element then those will come through.
GS.stringToElement Document Example:
For this example you will have to open the Javascript console for your browser.

When you click the "This Document" button you are creating the element in the scope of the current document. Meaning that for example a <gs-text> element will activate because the gs- elements have been registered in this document. To tell if the element has activated: open it up in the console if there is an input element inside it then it has activated.

When you click the "Custom Document" button you are creating the element in the scope of the iframe which doesn't have the gs- elements registered in it. Meaning that the gs-text element will not activate.
