Mouse Position Normalizer Function

 Function:
GS.mousePosition(<EVENT-OBJECT>);
Description:
This function makes getting the mouse position work the same on touch and non-touch devices.

In this example we are updating the "mouse-position" div whenever a mousemove event is caught. We are also preventing default and stopping propagation on mousedown so that touch devices don't scroll in this example.
